# Mini E-commerce API

>A mini e-commerce API built using Node, Express and MongoDB

#### Installation
* run   ```git clone https://github.com/Young-Einstein10/mini_ecommerce_api``` 

## :gear: Installation and running server

#### npm
    npm install
    nodemon server.js

OR

#### yarn
    yarn install
    nodemon server.js

#### Start Server
* run	```npm run dev``` or ```yarn run dev```
